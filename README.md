# Steps to run the applications
## Prerequisites
Before setting up the project, you have the following installed:
1. Install Docker Desktop [Download Docker Desktop](https://www.docker.com/products/docker-desktop/)
2. Install Pgadmin 4  [Download Pgadmin 4](https://www.pgadmin.org/download/)
3. Install Golang [Download Golang](https://go.dev/doc/install)

## Steps to run the application
1. Clone [Repository](https://github.com/athul-narayanan/course_tracker_api.git) locally.
2. All Environment variables are added in the docker-compose.yml to run in local
3. Application can be run by executing ```docker-compose down -v && docker-compose up --build```
